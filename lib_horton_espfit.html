horton.espfit.cost.setup_weights(system, grid, dens=None, near=None, far=None)¶

Define a weight function for the ESPCost

Arguments:

system
The system for which the weight function must be defined
grid
A UniformGrid object.

Optional arguments:

dens
The density-based criterion. This is a three-tuple with rho, rho0 and alpha.
near
Exclude points near the nuclei. This is a dictionary with as items (number, (R0, gamma)).
far
Exclude points far away. This is a two-tuple: (R0, gamma).
